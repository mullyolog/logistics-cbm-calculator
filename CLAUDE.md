# CBM 계산기

## 개요
화물 크기·수량 입력 → CBM, 컨테이너 적재량, R/T(해상)·C/W(항공) 자동 계산.
제작: 물류로그 (자격증 5개 — 물류관리사·국제무역사·보세사·원산지관리사·유통관리사 2급)

## 파일 구조
- index.html 1개 (HTML + CSS + JS 모두 내장)
- 외부 라이브러리 사용 금지

## 디자인
- 컬러: #1a365d (메인) / #234e7e (보조) / #e2a84b (강조) / #f7fafc (배경) / #e53e3e (위험) / #2e7d32 (성공)
- 폰트: 'Pretendard', -apple-system, BlinkMacSystemFont, sans-serif
- 레이아웃: 카드형, max-width 800px, 중앙 정렬
- 모바일: 반응형, 터치 영역 44px×44px 이상
- 입력: inputmode="decimal" (가로/세로/높이/무게), inputmode="numeric" (수량)

## 입력
- 가로(cm), 세로(cm), 높이(cm), 수량(개), 무게(kg, 선택)
- 다종 박스: "행 추가" 버튼으로 여러 박스 동시 입력
- 모든 필드에 placeholder (예: "예: 50")

## 계산 공식
- 1박스 CBM = (가로 × 세로 × 높이) ÷ 1,000,000
- 총 CBM = 모든 박스 행의 (1박스 CBM × 수량) 합산
- 컨테이너 기준: 20ft = 28 CBM, 40ft = 58 CBM, 40HC = 68 CBM
- 최대 박스 수 = floor(컨테이너CBM ÷ 1박스CBM)
- 필요 컨테이너 수 = ceil(총CBM ÷ 컨테이너CBM)
- 1박스 CBM > 28 → "20ft 적재 불가 (1박스 > 28CBM)" 경고
- 무게 입력 시:
  · 총 중량(kg) = 무게 × 수량 (모든 행 합산)
  · 총 중량(톤) = 총 중량(kg) ÷ 1000
  · 해상 R/T = max(총 CBM, 총 중량 톤)
  · 항공 C/W = max(총 중량 kg, 총 CBM × 167)

## 에러 처리 (필수)
- 빈값: "[필드명] 값을 입력하세요"
- 음수: "양수를 입력하세요"
- 비숫자: "숫자만 입력하세요"
- 0 입력 (가로/세로/높이/수량): "[필드명] 값을 입력하세요"

## UI 요소
- 행 추가 / 행 삭제 (행 2개 이상일 때만 표시)
- 초기화 (모든 입력 클리어)
- 결과 복사 (총 CBM·컨테이너 수 등 텍스트로 클립보드)
- URL 복사 (입력값을 URL 파라미터로 인코딩 후 클립보드)

## 차별화 기능 (필수 구현)

### 1. 컨테이너 시각화
20ft·40ft·40HC 각각 적재율 막대바 표시. CSS만으로 구현 (외부 차트 라이브러리 X).
표시 형식 예: "20ft 컨테이너  ▓▓▓▓▓▓▓░░░ 67%  (필요 1대)"
- 막대바: div 2개 (배경 + 채워진 부분)
- 100% 초과 시(1대로 부족) 빨간색 + "다중 컨테이너 필요" 표기
- 1박스 > 컨테이너CBM이면 "적재 불가" 표시

### 2. 계산 방법 토글 (HTML <details> 태그)
<details><summary>▼ 계산 방법 보기</summary> 안에:
- CBM 공식: "(가로 × 세로 × 높이) ÷ 1,000,000"
- R/T 공식: "해상 운임은 부피(CBM)와 중량(톤) 중 큰 값으로 계산. R/T = max(총 CBM, 총 중량 톤)"
- C/W 공식: "항공 운임은 부피와 중량 중 큰 값. 부피 환산 비율 167 적용. C/W = max(총 중량 kg, 총 CBM × 167)"
- 컨테이너 기준: "20ft 28 CBM / 40ft 58 CBM / 40HC 68 CBM (실용 적재량 기준, 화물 특성에 따라 차이)"

### 3. URL 공유
- 다종 박스 인코딩 형식: ?rows=가로,세로,높이,수량,무게;가로,세로,높이,수량,무게
  · 예: ?rows=50,40,30,100,15;120,80,100,10,0
  · 무게 미입력 시 0 또는 빈 값
- "URL 복사" 버튼 → 현재 입력값 반영된 URL을 클립보드 복사
- 페이지 로드 시 URL 파라미터 있으면 자동으로 입력값 채우고 계산
